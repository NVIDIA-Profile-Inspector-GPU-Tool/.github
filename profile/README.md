# NVIDIA Profile Inspector GPU Tool

**NVIDIA Profile Inspector GPU Tool** is a powerful utility that exposes hundreds of hidden driver settings not available in the standard NVIDIA Control Panel. As an advanced **NVIDIA Driver Settings Editor**, you can fine-tune anti-aliasing modes (MSAA, SSAA, MFAA), anisotropic filtering levels, texture filtering quality, ambient occlusion, and power management behavior. This **NVIDIA Game Profile Manager** lets you create, modify, and export per-game driver profiles to optimize performance or visual quality for specific titles. With **NVIDIA Hidden Tweaks Tool**, you can enable SLI compatibility bits, resizeable BAR (ReBAR), frame rate limiters, vertical sync tear control, and VR pre-rendered frames.
